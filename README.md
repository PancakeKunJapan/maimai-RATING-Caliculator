# maimaiでらっくす RATING計算機
maimaiでらっくす 譜面定数と達成率からレートを計算します｡
使い方
1. 達成率%を入力してください｡101.0000から0.0000の中で入力してください｡小数部分は簡略化しても大丈夫です｡(100.5や98.76など)
2. 譜面定数の整数部分を選択してください｡(12.3なら12､14.0なら14など)
3. 譜面定数の小数部分を選択してください｡(12.3なら3､14.0なら.0)
4. AP済みならチェックボックスにチェックを入れてください｡
単曲レートは一曲のレート値が表示されます｡RATINGは単曲レート*50曲分のレート値が表示されます｡
計算は自動で行われます｡
